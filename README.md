# mini-SPA-de-normas-b-sicas-del-Reglamento-del-Aprendiz-SENA
Realizar una mini SPA en React sobre normas básicas del Reglamento del Aprendiz SENA (en especial, uso del uniforme, carné visible, puntualidad, seguridad y convivencia). El objetivo es practicar componentización, props, estado (useState), eventos, composición y pensamiento declarativo.
